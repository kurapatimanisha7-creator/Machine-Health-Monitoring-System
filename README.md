# Machine-Health-Monitoring-System
The Machine Health Monitoring System is an IoT and AI-based project designed to monitor the health of industrial machines in real time. The system collects data from sensors such as temperature, vibration, and sound sensors, then analyzes it using Machine Learning algorithms to detect abnormal conditions and predict machine failures before they occur.

This project helps industries reduce unexpected breakdowns, minimize maintenance costs, improve machine reliability, and increase productivity through predictive maintenance.

Features
📊 Real-time machine health monitoring
🌡️ Temperature and vibration monitoring
🤖 AI/ML-based fault prediction
🚨 Automatic fault alerts
📈 Performance dashboard
💾 Data storage using MySQL
🔧 Predictive maintenance support
Technologies Used
Python
Flask
HTML, CSS, JavaScript
MySQL
IoT Sensors
Machine Learning
Objective

To develop an intelligent system that continuously monitors machine health, predicts failures in advance, and helps industries reduce downtime and maintenance costs.
